
# CmsModern

Een eenvoudige, moderne CMS-basis in ASP.NET Core + MySQL + Bootstrap.

## Installatie

1. Maak de database aan met `database.sql`
2. Vul je MySQL-gegevens in in `appsettings.json`
3. Start het project in Visual Studio Code of Visual Studio
4. Publiceer naar SmarterASP.NET

## Features

- Pagina-overzicht en bewerken
- Bootstrap styling
- Uitbreidbaar met gebruikersbeheer en meer
